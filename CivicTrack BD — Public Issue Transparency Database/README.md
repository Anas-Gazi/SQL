# CivicTrack BD - Public Issue Transparency Database

This repository contains SQL scripts for a civic issue tracking database focused on Bangladesh administrative regions.

## Project Purpose

The database models how citizens submit complaints, how officers process them, and how issue resolution is tracked with status history, evidence, and ratings.

## File Guide

- `CivicTrack BD`: Primary schema creation script (database + tables + indexes).
- `data.sql`: Realistic seed data for divisions, districts, users, complaints, and related entities.
- `main.sql`: Validation and exploratory queries to verify loaded data.
- `practice.sql`: Practice query examples for quick SQL exercises.
- `Practice Q-1`: Introductory SQL practice script using a simple employee table.

## Recommended Execution Order

1. Run `CivicTrack BD` to create the database schema.
2. Run `data.sql` to populate data.
3. Use `practice.sql`  for practice and learning.
